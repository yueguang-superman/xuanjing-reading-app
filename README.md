# 玄镜测算台

一个无外部依赖的前后端演示应用，用于娱乐性质的手相、面相和生辰八字测算。

## 启动

```bash
node server.js
```

浏览器打开：

```text
http://localhost:3000
```

## 发布给别人使用

本项目已包含公网部署配置：

- `render.yaml`：Render Web Service 配置。
- `Dockerfile`：服务器或云平台容器部署。
- `DEPLOY.md`：具体发布步骤。

部署到 Render、Railway、Fly.io、VPS 等平台后，把平台生成的网址发给别人即可。

## 测试

```bash
node --test
```

## 功能

- 前端：输入姓名、性别、出生日期、出生时辰、出生地，上传手相和面相照片。
- 图片：浏览器本地提取亮度、对比、色温、对称、线纹密度等匿名特征值，不上传原图。
- 后端：生成八字四柱、五行分布、手相/面相娱乐解读、幸运信息和行动建议。

## 说明

本项目输出仅用于娱乐、文化体验和产品演示，不构成现实决策、健康、财务或关系建议。八字排盘使用公历和近似节气边界，若要做严肃传统命理研究，需要接入精确节气历法。
